# Ansh Sharma - Portfolio Website

A personal portfolio website built with **HTML**, **CSS**, and **JavaScript** as part of a BCA web development assignment. The site showcases my skills, projects, hobby, and contact information with a fully responsive, modern blue-themed design.

## Features

### HTML and CSS (Assignment 1)
- Responsive layout working on mobile, tablet, and desktop
- Custom Google Fonts: **Syne** (display) and **DM Sans** (body)
- Animated hero section with floating avatar and spinning gradient ring
- CSS Grid layout for the Projects section
- Flexbox for navigation, about section, and contact layout
- Smooth scroll with active nav link highlighting
- Sticky navbar that gains background on scroll

### JavaScript (Assignment 2)
- **Hobby Interaction** - Add new benefits via input box (Enter key also works); each item has a Delete button with smooth remove animation
- **Contact Form Alert** - Form submission triggers a browser alert with the users name and email, plus an inline success message
- **Dynamic Footer Clock** - Live date and time in the footer, ticking every second in the format: Tuesday, 03 March 2026, 14:35:22
